# Short Treasure Hunt Game (Python)

This is a short, text-based treasure hunt game built using Python.  
The game runs in the command line and allows the player to explore locations, collect items, and find hidden treasure.

---

## Game Overview

The player explores three locations:

1. **Old Dock**
   - Try to open a locked box
   - 70% chance to get the **OLD KEY**
   - Find a clue on a note

2. **Ancient Ruins**
   - Solve a riddle
   - Correct answer gives the **COMPASS**

3. **Dig Site**
   - Use the **OLD KEY** and **COMPASS** to find the treasure
   - Or dig randomly with a small chance of winning

To win the game, the player must:
- Collect the **OLD KEY**
- Collect the **COMPASS**
- Use both at the Dig Site

---

## How to Run the Game

1. Save the file as `short_treasure_hunt.py`
2. Make sure Python is installed
3. Run the game using:

```bash
python short_treasure_hunt.py
